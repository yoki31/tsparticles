Properties

blink

blink: boolean
  • Defined in src/Options/Interfaces/Particles/Links/ILinks.ts:15

Enables random blinking, if set to true and color is set to "random" the link will have a random color each frame resulting in a blinking animation

color

color: string | IColor
  • Defined in src/Options/Interfaces/Particles/Links/ILinks.ts:23

Link line color, can be a string or an IColor object. If the value is "random", if blink is true sets a random color each frame, if consent is true a random color is used for all links, if blink and consent are false the color will be a gradient with the two linked particles color

consent

consent: boolean
  • Defined in src/Options/Interfaces/Particles/Links/ILinks.ts:28

If enabled the links will have a single random color, if color value is "random"
